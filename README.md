# AutoWinMerge: WinMerge 自動操作

- Win32アプリの自動操作の学習プログラム
- Appium (+ WinAppDriver) で自動操作を実現しています


# 環境構築

## WinMerge

- https://winmerge.org/downloads/ から取得

## WinAppDriver

- https://github.com/Microsoft/WinAppDriver から取得します

## Appium

- https://github.com/appium/appium から取得します
